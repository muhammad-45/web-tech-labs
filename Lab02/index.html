Lab 02

The Lab 02 project is a multi-page website that shows the basic use of HTML and CSS. All pages are connected with a common navigation bar so the user can easily move between different tasks.

The profile page presents information about Muhammad Saad Afridi, a Computer Science student at UET Peshawar. The schedule page displays a weekly timetable using tables with a highlighted lunch break. The navigation page contains internal and external links including the UET website.

Task 4 includes a registration form and a contact form. The contact form is connected with Formspree for email submission. The same color theme and simple styling are used on all pages to keep the design clean and consistent.
